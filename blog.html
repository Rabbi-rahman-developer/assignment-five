1. What is the Document Object Model (DOM)?

The DOM is a programming interface that allows scripts to update the content, structure, and style of a document.

2. How do you select an element from the DOM?

You can select an element using methods like document.getElementById, document.querySelector, etc.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the practice of using a single event listener to manage events for multiple elements, improving performance.

4. How do you manipulate an element's attributes and styles using the DOM?

You can use methods like element.setAttribute() to change attributes and element.style.property to change styles.
